# Origin Fledgling

A standalone **Vampire Fledgling** origin datapack for the Minecraft Origins mod.

## Requirements

- Minecraft Java Edition with the Origins mod installed
- Pehkui, used for the fledgling size-scaling power
- A Minecraft version compatible with the `pack_format` in `pack.mcmeta`  
  This pack currently uses `pack_format: 15` for Minecraft 1.20/1.20.1. If you install it on another version, update `pack.mcmeta` to the appropriate pack format.

## Installation

1. Download or clone this repository.
2. Place the repository folder, or a zip of its contents, in your world's `datapacks` folder.
3. Run `/reload` or restart the world/server.
4. Choose **Vampire Fledgling** from the normal Origins origin layer.

## Notes

The datapack appends the origin to the standard Origins layer with `replace: false`, so it should not replace the default origin list.
